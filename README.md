
# Smart curtain
智能窗帘

## Smart curtain有哪些功能？
* 基于arduino平台
* 能够采集环境的状态如温度和光照强度
* 可以将状态信息实时显示到小程序和Oled屏幕中
* 有配套的微信小程序，可以设置手动模式和自动模式
